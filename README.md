# MoneroWallet

A secure, self-hosted crypto wallet that runs entirely in your browser. No servers. No accounts. Your keys, your coins.

## Features

- Non-custodial  your seed phrase never leaves your device
- Multi-chain  supports Ethereum, BNB Smart Chain, and more
- Live prices  real-time token balances and market data
- Transaction history  full on-chain activity powered by Moralis
- Dark and light mode  clean UI that adapts to your preference
- Mobile-first  designed for mobile, works great on desktop
- Open source  fully auditable, self-hostable, fork-friendly

## Getting Started

```bash
npm install
npm run dev
npm run build
```n
Open http://localhost:5173 in your browser.

## Self Hosting

Build and drop the dist/ folder onto any static host  Netlify, Vercel, Cloudflare Pages, or your own server.

## Tech Stack

- React + Vite
- Tailwind CSS
- ethers.js
- Zustand
- Moralis API for transaction history
